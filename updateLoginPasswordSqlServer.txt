Pour modifier le nom d'utilisateur (login) et le mot de passe d'un utilisateur SQL Server, vous pouvez suivre ces étapes en utilisant SQL Server Management Studio (SSMS) ou en utilisant des scripts SQL.

À l'aide de SQL Server Management Studio (SSMS) :

Lancez SQL Server Management Studio.

Connectez-vous au serveur SQL Server en utilisant un compte ayant les autorisations nécessaires pour effectuer ces modifications.

Dans l'Explorateur d'objets, développez le nœud "Sécurité" pour voir les utilisateurs.

Cliquez avec le bouton droit de la souris sur l'utilisateur que vous souhaitez modifier, puis sélectionnez "Propriétés".

Dans la fenêtre des propriétés de l'utilisateur, vous pouvez modifier le nom d'utilisateur dans l'onglet "Général" et le mot de passe dans l'onglet "Général" ou "Options de connexion". Entrez le nouveau nom d'utilisateur et le nouveau mot de passe, puis cliquez sur "OK" pour enregistrer les modifications.

Assurez-vous de redémarrer le service SQL Server pour que les modifications prennent effet.
